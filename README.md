# Sistem Bancar în Python

Aplicație interactivă în Python pentru gestionarea fluxurilor financiare ale unui cont bancar, dezvoltată prin structuri de date dinamice (liste și dicționare), module funcționale pentru filtrarea și sortarea datelor, sisteme de rollback (Undo) bazate pe istoricul stărilor și mecanisme riguroase de validare a inputului.
###  Ce știe să facă aplicația:
- Crearea de conturi bancare cu un sold inițial.
- Depunere și retragere de bani (cu validare: nu poți extrage mai mult decât ai).
- Afișarea soldului curent în consolă.

###  Tehnologii folosite:
- Python 3
